Apple Watch Proyect

Strategic Goal

Create a simple and responsive application that helps students find their desired book and works properly on an apple watch. The application will be intuitive and easy to use for any student and/or teacher.

Relevant Questions

Questions:

  1. What kind of interface would you like the application to have?
  2. Which functionalities would you like the application to have?

Answers:

  • I would prefer a simple interface, I like download the app and know what to do from the begining. I don't like to spend time trying to figure out what I have to do. I would like and intuitive interface that is not hard to use and that lets you find books fast.
  • Given that the application will work inside an apple watch, I wouldn't like to type anything, since it's hard to type anything. It should use the voice recognition functionality included in the apple watch to let you search for books.

Interaction Methods

The application will be designed to be responsive and let the user have simple and quick interactions with it to have the best possible experience and show results fast. The interaction methods that will be used proyect are as follow:

  • Intuitive touch screen that will let the user navigate throughout the application.
  • The voice recognition that will help the user search for books.
