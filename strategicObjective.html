Strategic Objective

Create a responsive and user-friendly website that replaces the actual itesm library webpage by allowing the users to search books in a fast and easy way.

Tasks

To make a webpage that can compete with the itesm library site won't be easy, so here are the tasks that we must perform in order to achieve our goals.

  • Make a responsive and easy to use webpage.
  • Come up with an original idea that can solve the problematic.
  • See users' reactions to our product by releasing early prototypes.
  • Make sure every student that uses the library knows about the new website.

Steps

Next, we present the steps the user has to take in order to search for a book using our solution.

  1. User logs in with their id and password.
  2. The user now has access to the data bases and services that the library provides.
  3. User searches for a book in any of the data bases.
  4. The webpage displays all the results that match the user's search.
  5. Once the user selects an option, the webpage will display the book's details and show a map where the user can find it.
