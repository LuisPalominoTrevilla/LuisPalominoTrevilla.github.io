Strategic Objective

Find a book in the library inventory.

Tasks Steps 1 Steps 2 Steps 3

Tasks

1. Window with a search box for books by Author, Name or Keyword

2. Select the book you're searching for

3. Search the book at the library

Steps 1

1. Registered with your lastname, id and nip code

2. Once registered it gives you access to the books inventory in any ITESM library

Steps 2

1. Write the book you're searching for in the search box

2. Click the link to the book you desire

3. Deploys in which campus you can find it and the code to search the book inside the campus

Steps 3

1. Copy the book code

2. Search for the section of the book code

3. Wonder trough the hole section until you find your book
